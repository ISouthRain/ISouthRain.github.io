Corne无线分体键盘构建指南

 2024.3.28  2024.3.28

预览

文章适用人群

想要构建一个 无线 分体键盘, 并且当前还没有购买任何 构建键盘所需物品, 还处于研究如何构建阶段

Corne 键盘 组装起来非常简单, 至少 需要购买的物品 都可以在 中国淘宝 买的到

本文并不是一步一步教你从头开始 组装键盘, 而是对 GitHub - foostan/crkbd: Corne keyboard, a split keyboard with 3x6 column stag… 项目 增加 无线(蓝牙) 组装 补充

关于如何构建 Corne 键盘 你应该看官方文档

项目说明

个人构建Crone键盘使用的方案: Corne cherry V3

(这是必须的)键盘固件使用的是: Hello from ZMK Firmware | ZMK Firmware

QMK 固件不支持无线(蓝牙)

键盘布局推荐: GitHub - ISouthRain/miryoku_zmk: Miryoku is an ergonomic, minimal, orthogonal…

其它 Corne 方案应该也是可以的

有关续航

不算 休眠, 睡眠 浪费的电, 不开 LED, 不开RGB: 左边(带OLED): 96个小时一直使用时间, 所以它一定高于 96个小时

右边(不带OLED): 124个小时一直使用时间, 所以它一定高于 124个小时

20天续航是没有什么问题的

重要物件变更说明(必读)

无线 指的是 蓝牙

  1. 无论 ProMicro 还是 Elite-C 都是有线的 不应该购买这两个, 你应该购买 支持蓝牙的 主控制板

    例如: nrf52840

  2. 以下器件变更 只适用于 亚克力 板当外壳的

    1. Spacer M2 7.5mm 和 Spacer M2 9mm 金属柱 应该换成 M2 7mm 和 M2 12mm 尺寸

    2. 除此之外, 还必须购买电池, 每半个 Corne 键盘大概可以放下三个电池: 厚度只能最厚 5mm

      1. 401030-130毫安 厚度: 4mm 宽度: 10mm 长度: 30mm
      2. 505065-2500毫安 厚度: 5mm 宽度: 50mm 长度: 65mm
      3. 503040-600毫安 厚度: 5mm 宽度: 30mm 长度: 40mm

    3. 这是一个建议: 你可以在 PCB板 多开两个孔, 用于 电池供电, 因为这样会比较优雅, 当然修改 PCB 需要一定电路版知识

      这是建议性的, 不是必须的, 如果你没有修改两个孔, 电池供电看来是这样的:

      如果要开孔, 应该在 黄线 位置, 就是 主控制板 的最前面部分的下面

    4. 焊铁, 应该买个圆圆尖尖的, 这样可以将 亚克力板 开一个洞, 它是用于构建键盘必须的 焊铁链接

使用 焊铁 手动给亚克力板开孔, 用于支撑 PCB 板子 地下放置电池

如何给电池充电??

直接用 USB 数据线 插入 主控制板 口 即可
